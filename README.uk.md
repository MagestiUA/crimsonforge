# CrimsonForge

[🇬🇧 English](README.md) | 🇺🇦 Українська

CrimsonForge — повноцінна студія моддингу для **Crimson Desert**.

Дозволяє переглядати ігрові архіви, перекладати локалізацію за допомогою ШІ, переглядати та замінювати меші, замінювати аудіо, генерувати озвучку діалогів, патчити шрифти й записувати все назад із коректними контрольними сумами так, щоб гра запускалась як завжди.

CrimsonForge проводить дані через повний конвеєр:

`розшифрування (ChaCha20) -> розпакування (LZ4) -> парсинг -> зміна -> перепакування -> повторне шифрування -> оновлення PAMT -> оновлення PAPGT -> готово до гри`

## Основні можливості

### Оглядач архівів (Explorer)

- Перегляд понад **1,4 мільйона** файлів у ігрових пакетах зі швидким фільтруванням
- Попередній перегляд текстур, мешів, аудіо, тексту, шрифтів та веб/UI-файлів
- Вбудований редактор для CSS, HTML, XML, JSON та даних локалізації
- Витягування файлів з автоматичним розшифруванням і розпакуванням
- Дії через контекстне меню для експорту/імпорту мешів, аудіо та патчингу в гру

### Моддинг мешів

- Перегляд та експорт мешів `.pac`, `.pam`, `.pamlod`
- Експорт у **OBJ** та **FBX**
- Імпорт відредагованих OBJ-файлів назад у гру
- Повноцінний round-trip для статичних мешів і редагування PAC-мешів зброї/персонажів
- Імпорт OBJ + патч у гру в один клік
- Пошук за назвою предмета в Explorer для швидкого знаходження потрібного асета

### Моддинг аудіо

- Перегляд, пошук, відтворення, експорт і заміна ігрового аудіо
- Прив'язаний текст діалогів для значної частини озвучення
- Експорт у WAV/OGG
- Імпорт WAV і патч назад у гру
- Підтримка перебудови WEM через Wwise

### ШІ Text-to-Speech

- Генерація заміни голосів для реплік діалогів
- Підтримка кількох TTS-провайдерів: Edge TTS, OpenAI, ElevenLabs, Google Cloud, Azure Speech, Mistral Voxtral
- Генерація та патчинг аудіо в один робочий процес

### Робоче середовище перекладу

- Парсинг і робота з понад **172 000** рядків локалізації
- Сховище проєкту на SQLite — швидке повне збереження, захищений від збоїв поточковий запис під час пакетного перекладу, автоматична міграція зі старих JSON-проєктів
- Пакетний ШІ-переклад з підтримкою кількох провайдерів
- Пакетні запити у форматі JSON-масиву (багато рядків за один виклик) з автоматичною бісекцією (розбиттям навпіл) при недовірі до відповіді
- Дедуплікація — однакові вихідні рядки перекладаються один раз і застосовуються до всіх входжень
- Автоматична обробка помилок 429/5xx окремо для кожного провайдера (очікування при перевищенні ліміту запитів vs повтор при тимчасовій помилці сервера)
- Система промптів, що зберігає плейсхолдери, теги та ігрову термінологію; транслітерує імена в алфавіт цільової мови, якщо він відрізняється від джерела
- Незалежний вибір цілі патчингу — перекладати з однієї мови гри, а результат патчити в інший мовний файл (корисно, якщо в грі немає рідного файлу для твоєї цільової мови)
- Підтримка глосарія
- Автозбереження та відновлення сесії
- Виявлення оновлень гри та злиття зі зміненими даними локалізації, включно з автоматичним поверненням у чергу рядків зі зміненим оригіналом

### Font Builder

- Витягування ігрових шрифтів
- Аналіз відсутнього покриття гліфів для цільових мов
- Підтягування потрібних гліфів з донорських шрифтів
- Патчинг оновлених шрифтів назад у гру

### Ship to App

- Генерація інсталяційних пакетів для перекладачів і мод-команд для кінцевих користувачів
- Збірка пакетів "запустив і працює" з `install.bat` та `uninstall.bat`
- Кінцевим користувачам не потрібні Python чи інструменти моддингу

### Patch to Game

- Автоматичне резервне копіювання перед патчингом
- Перебудова архівів і ланцюжків контрольних сум
- Безпечніша робота з пропатченими або частково модифікованими інсталяціями порівняно зі старішими інструментами

## Встановлення

### Варіант 1: Готова збірка

Якщо просто хочеш запустити CrimsonForge — візьми готовий виконуваний файл зі сторінки **GitHub Releases**.

- Завантаж `CrimsonForge.exe`
- Запусти напряму
- Окремий Python не потрібен

### Варіант 2: Запуск із коду

Вимоги:

- Python `3.12+`
- Crimson Desert, встановлений через Steam
- Хоча б один API-ключ, якщо потрібен ШІ-переклад або TTS

Кроки:

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Переклад-моди для кінцевих користувачів

CrimsonForge також підтримує робочий процес "перекладач -> гравець".

Перекладач може згенерувати пакетований реліз мода, а кінцевому користувачу потрібно лише:

1. Завантажити ZIP-пакет перекладача
2. Розпакувати будь-де
3. Двічі клацнути `install.bat`
4. Запустити Crimson Desert

Для видалення згенерований пакет також підтримує деінсталяцію через перевірку цілісності файлів Steam.

## Збірка окремого EXE

Проєкт містить spec-файл PyInstaller для збірки під Windows:

```powershell
python -m PyInstaller --clean --noconfirm CrimsonForge.spec
```

Результат:

- `dist/CrimsonForge.exe`

Збірка включає:

- повну директорію `data`
- необхідні ресурси для роботи
- `core/pa_checksum.dll`
